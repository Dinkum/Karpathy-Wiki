# Karpathy-Wiki

> **Note:** 100% inspired by Karpathy → https://x.com/karpathy/status/2039805659525644595
> I just wanted to document my scaffold.

An everything wiki. Drop documents in, ask questions, get answers all with Codex as your interface. Works with literally whatever documents you care about!

---

## Requirements

- [OpenAI Codex](https://openai.com/codex)
- [Obsidian](https://obsidian.md) — for reading the wiki
- This repo

---

## How to Use

Got documents? Drop them in `raw/` and tell Codex you added something.

> Want to query the wiki? **Ask Codex!**
> Want a new article? **Ask Codex!**
> Want to ask a question with the wiki as a knowledge base? **Ask Codex!**
> Want an audit of gaps and contradictions? **Ask Codex!**
> Want to build a hybrid ranked semantic vector + FTS/BM25 search tool for Codex to search the wiki with? **Ask Codex!**
